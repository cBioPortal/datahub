DepMap 24Q4 readme


Clinical data: 
	•	Data was retrieved from ‘Model.csv’ file from the ‘DepMap Public 24Q4 All Files’.
	•	Samples ID: CCLE ID
	•	Samples without a CCLE ID: Cell line name used as Sample ID 
	
	
Mutation data: 
	•	Data was retrieved from ‘OmicsSomaticMutations.csv’ file from the ‘DepMap Public 23Q4 All Files’.
	•	Annotation with Genome Nexus pipeline 
	•	Profile IDs in original mutations file mapped using ModelCondition.csv
